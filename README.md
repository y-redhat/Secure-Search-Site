# Secure-Search-Site
A private web proxy powered by Cloudflare Tunnel, Flask, and Tor, with encrypted communication. Encrypts destination traffic and protects user privacy through authentication and SSRF countermeasures. Browsing activity is effectively hidden to bypass tracking and blocking. Protects privacy.
